💡 實驗原理

法拉第電磁感應定律：$\varepsilon = -N\cdot \frac{d\Phi}{dt}$。楞次定律：感應電流的方向總是反抗引起感應電流的磁通量變化。

📐 核心公式

庫侖定律

$F = kQ_1Q_2/r^2$

真空中兩個點電荷間的相互作用力

電場強度

$E = \frac{F}{q} = \frac{kQ}{r^2}$
$E = U/d$

描述電場強弱的物理量

歐姆定律

$I = U/R$

電流與電壓成正比，與電阻成反比

電功率

$P = UI = I^2R = \frac{U^2}{R}$

描述電流做功快慢的物理量

磁感應強度

$B = F/IL$（左手定則）

描述磁場強弱的物理量

洛倫茲力

$F = qvB\sin\theta$

運動電荷在磁場中受到的力

法拉第電磁感應定律

$\varepsilon = -N\cdot d\Phi/dt$

磁通量變化產生感應電動勢

交流電有效值

$I_{有效} = \frac{I_{最大值}}{\sqrt{2}}$
$U_{有效} = \frac{U_{最大值}}{\sqrt{2}}$

交流電的等效直流值

變壓器

$U_1/U_2 = n_1/n_2$

原副線圈電壓比等於匝數比

電場能量

$W = \frac{1}{2}QU = \frac{1}{2}CU^2$

電容器儲存的能量

電阻串聯

$R_{总} = R_1 + R_2 + ...$

總電阻等於各電阻之和

電阻并聯

$1/R_{总} = 1/R_1 + 1/R_2 + ...$

總電阻的倒數等於各電阻倒數之和

⚖️ 重要定律

庫侖定律

真空中兩個點電荷間的作用力與它們的電荷量的乘積成正比，與它們距離的平方成反比，作用力的方向沿著它們的連線。

歐姆定律

導體中的電流與加在導體兩端的電壓成正比，與導體的電阻成反比。這是電路分析的基礎定律。

法拉第電磁感應定律

閉合電路中感應電動勢的大小，與穿過這一電路的磁通量的變化率成正比。$\varepsilon = -N\cdot \frac{d\Phi}{dt}$

楞次定律

感應電流的方向總是使感應電流產生的磁場阻礙引起感應電流的原磁場的變化。簡言之：來拒去留。

左手定則

伸开左手，使大拇指與四指在同一平面內並互相垂直，讓磁線穿過掌心，四指指向電流方向，則大拇指所指的方向即為通電導線所受安培力的方向。

右手定則

伸开右手，使大拇指與四指在同一平面內並互相垂直，讓磁線穿過掌心，大拇指指向導線運動方向，則四指所指的方向即為感應電流的方向。
