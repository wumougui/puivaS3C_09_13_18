📐 核心公式

勻變速直線運動

$v = v_0 + at$
$x = v_0t + \frac{1}{2}at^2$
$v^2 - v_0^2 = 2ax$

適用於勻變速直線運動的所有情況

自由落體運動

$v = gt$
$h = \frac{1}{2}gt^2$
$v^2 = 2gh$

初速度為零，僅受重力作用

牛頓第二定律

$F = ma$

合外力等於質量與加速度的乘積

平拋運動

$x = v_0t$
$y = \frac{1}{2}gt^2$
$\tan\theta = \frac{v_y}{v_x} = \frac{gt}{v_0}$

水平勻速，豎直自由落體

向心力（圓周運動）

$F = \frac{mv^2}{r} = m\omega^2r$
$a = \frac{v^2}{r} = \omega^2r$

指向圓心的合外力提供向心力

萬有引力

$F = GMm/r^2$
$v = \sqrt{GM/r}$
$T = 2\pi\sqrt{r^3/GM}$

天體運動的核心公式

功和功率

$W = Fs\cos\theta$
$P = \frac{W}{t} = Fv$

功是能量轉化的量度

動能定理

$W_{合} = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$

合外力做功等於動能變化量

機械能守恆

$E_1 = E_2$
$\frac{1}{2}mv^2 + mgh = 恆量$

只有重力或彈力做功時

動量定理

$I = Ft = mv_2 - mv_1$

衝量等於動量變化量

動量守恆定律

$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$

系統合外力為零時

彈性碰撞

$v_1' = \frac{(m_1-m_2)v_1}{m_1+m_2} + \frac{2m_2v_2}{m_1+m_2}$

動量、動能均守恆

⚖️ 重要定律

牛頓第一定律（慣性定律）

一切物體總保持勻速直線運動狀態或靜止狀態，除非有外力迫使它改變這種狀態。揭示了力的本質：力是改變物體運動狀態的原因，而不是維持運動的原因。

牛頓第二定律

物體的加速度跟所受的合外力成正比，跟物體的質量成反比，加速度的方向跟合外力的方向相同。$F=ma$是動力學的核心公式。

牛頓第三定律

兩個物體之間的作用力和反作用力大小相等、方向相反、作用在同一直線上。作用力和反作用力同時產生、同時消失。

機械能守恆定律

在只有重力或彈力做功的物體系統內，動能和勢能可以互相轉化，但機械能的總量保持不變。這是能量守恆定律在力學中的具體體現。

動量守恆定律

如果系統不受外力或所受合外力為零，系統的總動量保持不變。動量守恆是自然界最普遍、最基本的規律之一。

✏️ 練習鞏固

1 關於牛頓第一定律，下列說法正確的是

正確答案：C
牛頓第一定律表明，一切物體在不受外力時，總保持勻速直線運動狀態或靜止狀態。力不是維持運動的原因，而是改變運動狀態的原因。

2 一個物體從靜止開始做勻加速直線運動，第3秒內的位移是6m，則物體的加速度是

正確答案：B
第3秒位移 $\Delta x = v_2\cdot\Delta t + \frac{1}{2}a\Delta t^2 = 2a + 2 = 6\text{ m}$，得 $a = 2$ m/s²

3 關於功率，下列說法正確的是

正確答案：B
功率是描述做功快慢的物理量，功率越大表示做功越快。功率與做功多少、做功時間都有關（$P=\frac{W}{t}$），與機械效率無直接關係。

1 一質量為m的小球從高度為h處自由下落，碰到水平放置的固定彈簧上，將彈簧壓縮x後彈起。不計空氣阻力，則

正確答案：D
機械能守恆：初態重力勢能$mgh$ = 末態彈簧彈性勢能。小球在彈簧壓縮最大時速度為零、加速度最大（大於$g$）；彈起高度不會超過$h$。

2 水平面上有兩個質量分別為m₁和m₂的物體，它們之間的動摩擦因數為μ，相互作用力F作用在m₁上。要使兩物體保持相對靜止，F的最大值是

正確答案：D
兩物體相對靜止時，加速度相同。對整體分析：$F = (m_1+m_2)a$
對$m_2$分析：靜摩擦力提供加速度，$f = m_2a \leq \mu mg$
聯立解得：$F \leq \mu g(m_1+m_2)$

（2024高考）一顆衛星繞地球做圓周運動，軌道半徑為r，週期為T。已知地球半徑為R，則地球質量M為

正確答案：A
萬有引力提供向心力：$GMm/r^2 = m(2\pi/T)^2 r$
解得：$M = \frac{4\pi^2 r^3}{GT^2}$
